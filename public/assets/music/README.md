# Music Assets

Download the Happy Hardcore album from Lazer Dice:
https://lazerdice.bandcamp.com/album/happy-hardcore

Place the MP3 files here with these names:

- liftoff.mp3
- pirates.mp3
- happy-sphere.mp3
- boom.mp3
- light.mp3
- crono.mp3
- universal.mp3

The audio player will automatically pick them up.
